# WorkBuddy安装
# Windows 系统安装指南
## 一、下载

1. 访问 [WorkBuddy 官网](https://www.codebuddy.cn/work/)，点击 **立即下载**，等待安装包下载完成。

![](https://www.codebuddy.cn/docs/static/image.nuc5Wzyg.png)

## 二、安装

1. 双击已下载的安装包，启动安装程序；
    
    ![](https://www.codebuddy.cn/docs/static/image-1.Cv_fWdeI.png)
2. 勾选 **"我同意此协议"**，点击 **下一步**；
    
    ![](https://www.codebuddy.cn/docs/static/image-2.nX8V0i5W.png)
3. 选择安装路径（默认路径可直接点击下一步），点击 **下一步**；
    
    ![](https://www.codebuddy.cn/docs/static/image-3.C1dI3tJz.png)
4. 确认开始菜单文件夹，点击 **下一步**；
    
    ![](https://www.codebuddy.cn/docs/static/image-4.B1FHUmlf.png)
5. 建议勾选 **"创建桌面快捷方式"**，点击 **下一步**；
    
    ![](https://www.codebuddy.cn/docs/static/image-5.BlN7t4Co.png)
6. 确认配置无误，点击 **安装**；
    
    ![](https://www.codebuddy.cn/docs/static/image-6.BqsUqnLs.png)
7. 安装完成后，点击 **完成** 退出安装向导。
    
    ![](https://www.codebuddy.cn/docs/static/image-7.O4VQi_-H.png)

## 三、登录

1. 启动 WorkBuddy，点击 **登录** 按钮；
    
    ![](https://www.codebuddy.cn/docs/static/image-8.CK0afJ4M.png)
2. 勾选《服务条款》与《隐私协议》，使用微信扫码完成登录；
    
    ![](https://www.codebuddy.cn/docs/static/image-9.B-sCZ4xM.png)
3. 登录成功后自动返回客户端，即可开始使用。
    
    如需切换界面语言，点击左下角 **头像** → **语言** 进行切换。
    
    ![](https://www.codebuddy.cn/docs/static/image-10.Cyf1orgE.png)

## 四、版本更新

点击左下角 **头像** → **检查更新**，系统将自动检测版本。

![](https://www.codebuddy.cn/docs/static/image-11.DhncBaQi.png)

若已是最新版本则无需操作；若存在新版本，系统将自动下载并完成升级。
![](https://www.codebuddy.cn/docs/static/image-12.BL3Xs7uq.png)

# Mac 系统安装指南

## 一、下载

1. 访问 [WorkBuddy 官网](https://www.codebuddy.cn/work/)，点击 **下载 WorkBuddy**；
    
    ![](https://www.codebuddy.cn/docs/static/image-13.BpXsVrXL.png)
    
    **M 系列芯片** → 选择 `Mac ARM64` 版本；
    
    **Intel 芯片** → 选择 `Mac X64` 版本。
    
2. 在浏览器弹窗中允许下载，等待安装包下载完成。
    
    ![](https://www.codebuddy.cn/docs/static/image-14.BhZn1kDP.png)

## 二、安装

1. 双击已下载的 `.dmg` 磁盘映像文件，打开安装窗口；
    
    ![](https://www.codebuddy.cn/docs/static/image-15.sG7k4Kcf.png)
2. 将 **WorkBuddy** 图标拖入 **Applications** 文件夹；
    
    ![](https://www.codebuddy.cn/docs/static/image-16.2oor8hng.png)![](https://www.codebuddy.cn/docs/static/image-17.BxSolkQv.png)
3. 等待拷贝完成；
    
    ![](https://www.codebuddy.cn/docs/static/image-18.B8WzN7p9.png)
4. 右键推出安装磁盘，删除桌面残留的 `.dmg` 文件。
    
    ![](https://www.codebuddy.cn/docs/static/image-19.BnT816LK.png)

## 三、登录

1. 通过 **启动台** 或 `Command + 空格` 搜索打开 WorkBuddy；
    
2. 点击 **登录** 按钮，浏览器将自动打开登录页面；
    
    ![](https://www.codebuddy.cn/docs/static/image-20.CSF6dln-.png)
3. 勾选《服务条款》与《隐私协议》，使用微信扫码完成登录；
    
    ![467](https://www.codebuddy.cn/docs/static/image-21.BEKABQ55.png)
1. 登录成功后自动返回客户端，即可开始使用。
    
    ![](https://www.codebuddy.cn/docs/static/image-22.Da1Lz-Ry.png)

## 四、版本更新

点击左下角 **个人中心** → **检查更新**，系统将自动检测并完成升级。

![](https://www.codebuddy.cn/docs/static/image-23.C3iiE_yZ.png)


# 发起第一个任务

1. 进入 WorkBuddy 后，将看到新建任务页面，包含欢迎语、输入框和任务入口。
2. 在输入框中用一句话描述您的需求，例如：
    - "帮我把这份销售数据生成一个分析报告"
    - "整理这个文件夹里的图片，按日期重命名"
    - "根据这份需求文档生成一个 PPT"
3. 您可以选择 AI 模型。
4. 点击发送按钮或按回车键，WorkBuddy 即开始自主执行任务。
5. 您可以继续新建其他任务，让多个任务并行推进，并在侧边栏中随时切换查看。

![](https://www.codebuddy.cn/docs/static/new-task-page.B8hsXU1c.png)

## 一、界面介绍

### 1）工作模式

WorkBuddy 提供三种工作模式，对应不同的执行权限与交互方式：

![](https://www.codebuddy.cn/docs/static/image-1.CpdWAzFx.png)

| 模式             | 说明             | 适用场景           | 图片示例                                                           |
| -------------- | -------------- | -------------- | -------------------------------------------------------------- |
| **问一问（Ask）**   | 仅问答与信息查看，不修改文件 | 了解内容、确认需求      | ![](https://www.codebuddy.cn/docs/static/image-2.BSLOXG3U.png) |

首次使用优先选择 **Ask** 模式；熟悉后再按任务复杂度切换至 **Plan** 或 **Craft** 模式。处理重要文件前请先备份。

### 2）切换模型
WorkBuddy 内置多个主流模型，可按需灵活切换：
![666](https://www.codebuddy.cn/docs/static/image-5.CGnLKzX0.png)

|模型|适用场景|
|---|---|
|**MiniMax M2.5**|Excel 处理、数据分析、PPT 生成，执行速度快|
|**智谱 GLM-5**|多步骤、长流程的复杂任务|
|**Kimi K2.5**|截图分析、图片转文档等视觉类任务|
|**DeepSeek V3.2**|日常问答、文案撰写，响应快、成本低|
|**腾讯混元**|中文写作、会议纪要、中文文档处理|

### 3）设置工作目录
**使用前建议先设置工作目录。** WorkBuddy 将在该目录中读取和保存文件，未指定路径的任务也会优先在此目录内执行。
![413](https://www.codebuddy.cn/docs/static/image-6.BumYl1WH.png)

什么是文件路径？

路径即文件或文件夹在电脑中的位置，示例：
- **Windows**：`C:\Users\你的用户名\Desktop\工作资料`
- **Mac**：`/Users/你的用户名/Desktop/工作资料`

无需手动输入——设置时直接在文件选择器中定位并选中目标文件夹即可。若未提前指定，WorkBuddy 会在新建任务时自动创建独立目录。
如何选择合适的工作目录？
建议按任务类型分别建立目录，例如 `发票整理`、`周报`、`待处理照片` 等，便于管理并降低误操作风险。

### 4）选择技能
**技能（Skills）** 用于扩展 WorkBuddy 的任务能力。启用对应技能后，系统可完成更多特定类型的工作。
![694](https://www.codebuddy.cn/docs/static/image-7.C15-_p58.png)

在对话框中选择已安装技能，WorkBuddy 执行任务时将自动调用对应能力：

![598](https://www.codebuddy.cn/docs/static/image-8.D6W-HK-5.png)

|技能示例|效果|
|---|---|
|**PPT 制作**|根据数据生成汇报演示文稿|
|**海报生成**|根据需求生成活动海报|
|**自动化报表**|合并多份 Excel 并输出图表报告|

WorkBuddy 内置 **20+** 技能包，覆盖文档处理、数据报表、海报设计、文件整理等场景，同时兼容 **OpenClaw** 社区技能导入，也支持通过自然语言创建自定义技能。

## 二、新建任务

每个对话对应一个 **独立任务**，任务之间互不影响，各自维护独立的工作空间与上下文。
1. 在主界面点击 **新建任务**（若未提前指定目录，系统自动创建独立工作目录）；
2. 输入任务内容、选择工作目录与模型后发送，即可开始执行。

执行中的任务显示在左侧边栏，可随时切换查看进度。WorkBuddy 支持 **多任务并行处理**。

# Claw（远程任务）
简介
Claw 用于接收远程指令并查看执行记录，适合通过手机等外部渠道向本地 WorkBuddy 下发任务。

![](https://www.codebuddy.cn/docs/static/claw-1.zJpCjuXt.png)

## 一、功能说明
Claw 页面可查看远程任务的完整执行记录，包括思考过程、操作步骤、生成文件及最终结果。

### Claw 与普通任务的区别

| 对比项   | 普通任务      | Claw              |
| ----- | --------- | ----------------- |
| 工作目录  | 自由指定      | 固定使用 `Claw` 专属文件夹 |
| 新建对话  | 支持多个并行任务  | 仅一个会话，所有远程指令集中处理  |
| 上下文管理 | 可清空历史重新开始 | 保留完整对话历史，不可清空     |

本地操作优先使用主界面的普通任务；需要远程触发或统一查看记录时，再使用 Claw。

## 二、配置方式

1. 点击左下角 **个人中心** → **Claw 设置**；
    
    ![310](https://www.codebuddy.cn/docs/static/claw-2.CdC1G6hD.png)
1. 按需配置对应平台的集成接入。
    
    ![](https://www.codebuddy.cn/docs/static/claw-3.DStnIq9c.png)

# 连接器
一句话
连接器用于将 WorkBuddy 与外部服务进行对接，实现数据互通与能力扩展。通过配置连接器，您可以让 AI 助手直接访问和操作第三方服务。
## 一、什么是连接器
连接器是 WorkBuddy 与外部系统之间的桥梁。它基于标准化协议（如 MCP）将外部服务的能力引入到 AI 工作流中，让您无需手动切换工具即可完成跨平台操作。

| 场景       | 说明             | 示例          |
| -------- | -------------- | ----------- |
| **数据查询** | 从外部数据源获取信息     | 查询数据库、检索文档  |
| **服务调用** | 调用第三方 API 完成操作 | 发送邮件、创建日程   |
| **文件管理** | 访问云端存储与文件系统    | 读取网盘文件、上传附件 |
| **消息通知** | 与即时通讯工具集成      | 发送企业微信/飞书消息 |
## 二、进入连接器管理
在左侧导航栏中点击 **连接器**，即可进入连接器管理页面。页面展示所有可用的连接器，当前支持 **QQ邮箱** 和 **腾讯乐享**，并支持自定义连接器。
![连接器管理入口](https://www.codebuddy.cn/docs/static/connector-management-entry.CvbpOPfA.png)

## 三、连接 QQ 邮箱
QQ 邮箱连接器支持收发、搜索和整理 QQ 邮件，用自然语言读取邮件内容、汇总邮件线程、管理文件夹。
### 1）点击添加
在连接器管理页面找到 **QQ邮箱** 卡片，点击右侧的 **+** 按钮发起连接。
![点击添加 QQ 邮箱连接器|671](https://www.codebuddy.cn/docs/static/qq-mail-add-button.BDr1PCfF.png)
### 2）扫码授权
系统会跳转到 QQ 邮箱授权页面，显示二维码。使用 **QQ 邮箱 App**（7.1.5 / 鸿蒙 0.2.9 及更新版本）扫码进行授权。
![QQ 邮箱扫码授权页面|876](https://www.codebuddy.cn/docs/static/qq-mail-qrcode-auth.CpvZGV-q.png)

### 3）确认权限
在手机端确认 WorkBuddy 申请的访问权限，包括：
- **账号信息**：获取邮箱账号基础信息
- **读取邮件**：读取收件箱内近一个月的邮件
- **发送邮件**：代为发送邮件
- **删除邮件**（可选）：已删除邮件会移入「已删除」文件夹
确认无误后点击 **同意授权**。

![手机端确认授权权限|395](https://www.codebuddy.cn/docs/static/qq-mail-confirm-permission.DqVZH-62.png)

### 4）授权成功
手机端显示 **授权成功**，可在「设置 - 账号 - 安全管理 - 登录设备和授权码 - 应用授权」中管理授权应用。
![手机端授权成功提示|412](https://www.codebuddy.cn/docs/static/qq-mail-auth-success-mobile.BMvQVrZs.png)

### 5）连接完成
回到 WorkBuddy，QQ 邮箱卡片名称旁会出现绿色圆点，并提示 **连接器 QQ邮箱 已连接**，右侧显示启用/禁用开关。
![QQ 邮箱连接成功状态 - 刷新中|894](https://www.codebuddy.cn/docs/static/qq-mail-connecting.l5gGXLjA.png)![QQ 邮箱连接成功状态 - 已连接|885](https://www.codebuddy.cn/docs/static/qq-mail-connected.UTsl-Drg.png)

# 自动化
简介
自动化功能适用于周期性、重复性任务（如每日简报、周报汇总、定时数据整理）。配置完成后，WorkBuddy 按设定时间自动执行并将结果保存到指定目录。
## 一、创建自动化任务
### 1）打开自动化页面
可查看已安排的任务及历史执行记录。
![1047](https://www.codebuddy.cn/docs/static/auto-1.BR-oYIXJ.png)

### 2）添加任务
点击右上角 **添加**，填写以下配置项：
![](https://www.codebuddy.cn/docs/static/auto-2.WrD5blOe.png)

|配置项|说明|
|---|---|
|**名称**|用于区分不同自动化任务|
|**工作空间**|指定任务执行目录及文件保存位置，默认自动分配automation-claw-xxxx工作空间|
|**提示词**|描述任务目标和输出要求|
|**选择模型和技能**|指定执行任务的模型和技能|
|**定时规则**|设置执行频率和生效日期区间|
|**推送到小程序**|开启后，推送会通过安全链路把件同步到云端，以便在程序端能接收到数据|

示例
**每日 AI 新闻推送**
![661](https://www.codebuddy.cn/docs/static/auto-3.BkLK3JTY.png)

不想从零编写提示词？可直接使用 **任务模板**，覆盖新闻推送、周报生成、体检预约以及学习计划等常见场景，选择后按需修改即可。
